# Security and Update Policy

## Supported Versions

We have various versions, here are the supported and updated ones.

| Version | Supported          |
| ------- | ------------------ |
| pre-release versions (alpha, beta, release candidate) | :x: Unsupported - these are dev builds and may change at any time |
| v1.x.x | :heavy_check_mark: Fully supported - these are full releases and will be supported with both security and feature updates |
